Hyperspectral demosaicing

Hyperspectral demosaicing aims to recover full spectral information at each pixel from a mosaicked image captured using a snapshot camera. We demonstrate, that robust demosaicing models with near SOTA performance can be trained using existing RGB datasets with simple hyperspectral augmentations. Resulting model was used in multiple internal projects. Paper: Cube-it ... (2024)

Inspecting high-voltage towers using drones

Our group researched the use of hyperspectral imaging for the task of corrosion detection and corrosion severity estimation. We worked in cooperation with the Belgian electrical grid operator (Elia). Paper: Drone based ... (2024)

Hyperspectral object tracking

Can hyperspectral information improve object tracking performance? In theory, spectra add discriminative power which the tracker can use to distinguish between similar objects. To answer this question, multiple editions of hyperspectral object challenge were organized (some of which I co-organized). Paper: Helios ... (2023)
